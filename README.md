# Genova CLI

Simple CLI tool for scaffolding projects.

## Installation

```bash
npm install -g genova-cli
```

## Usage

```bash
genova
```

Follow the prompts to create your project.

## Features

- **Frontend**: Vite, Next.js
- **Backend**: Express, Hapi with TypeScript/JavaScript
- **Database**: PostgreSQL, MySQL, SQLite support
- **Fullstack**: Monorepo structure with frontend and backend
